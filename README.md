<h2> About the Project</h2>
<p>
  <b>Saladverse</b> is a premium wellness-focused website designed to redefine healthy eating.
  It offers a luxurious digital experience where users can explore curated salads, 
  customize their meals, and embrace a healthier lifestyle.
</p>

<h2> Features</h2>
<ul>
  <li>Elegant and modern UI design</li>
  <li>Custom salad builder experience</li>
  <li>Fully responsive across devices</li>
  <li>Fast and optimized performance</li>
  <li>Smooth animations and premium feel</li>
</ul>
